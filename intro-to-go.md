# WHAT IS GO?
Go is a cross-platform, open source programming language
Go can be used to create high-performance applications
Go is a fast, statically typed, compiled language known for its simplicity and efficiency
Go was developed at Google by Robert Griesemer, Rob Pike, and Ken Thompson in 2007
Go's syntax is similar to C++
# WHAT IS GO USED FOR ?
    Web development 
    Developing network-based programs
    Developing cross-platform enterprise applications
    Cloud-native development
# WHY GO IS USED ?
Go is fun and easy to learn
Go has fast run time and compilation time
Go supports concurrency
Go has memory management
Go works on different platforms 
# SOME CHARACTERISTICS OF GO
Go is statically typed
Fast runtime
Easily compiled
Fast compile time 

# WHAT IS CONTAINED IN A GO FILE
A Go file consists of the following parts:

    Package declaration
    Import packages
    Functions
    Statements and expressions

```go
    package main   // package declaration
    import ("fmt") // imports package

func main() { // main function
  fmt.Println("Hello World!") // statement and expression
}
```
A code can be written more compact like this :
```go
package main; import ("fmt"); func main() { fmt.Println("Hello World!");}
```
