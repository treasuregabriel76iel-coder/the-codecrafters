# GO DATA TYPES
Go has three basic data types :
1. bool (true or false)
2. numeric (integers)
3. string (represents a string value)

# Boolean Data Types
A boolean data type is declared with the bool keyword and can only take the values true or false.

# Integer Data Types
Integer data types are used to store a whole number without decimals, like 5 or -50.
Integer could be signed integers (can store negative and positive numbers) and unsigned integers (can only store non-negative values)

# Float Data Types
The float data types are used to store positive and negative numbers with a decimal point, like 5.3, -2.34, 1234.4321. The float data types have 2 keywords : float32 and float64. the number attached to the end of the word shows the bitsize.

# String Data Types
The string data type is used to store a sequence of characters (text). String values 
must be surrounded by double quotes.
